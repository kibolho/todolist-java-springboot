# todolist java springboot

### Install packages Maven

```bash
mvn clean install
```

### Run project

```bash
mvn spring-boot:run
```


### Database

localhost:8080/h2-console

Driver Class:	org.h2.Driver
JDBC URL: jdbc:h2:mem:todolist
